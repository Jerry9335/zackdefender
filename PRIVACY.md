# Zack Defender 隐私策略

**最后更新：2026 年 7 月 7 日**

Zack Defender（以下简称"本软件"）重视你的隐私。本隐私策略说明了本软件如何收集、使用和保护你的数据。

---

## 1. 我们收集的数据

### 1.1 我们**不**收集的数据

本软件在设计上尽可能减少数据收集。我们**不**收集以下任何数据：

- **个人身份信息**：不收集姓名、电子邮件、电话号码、地址、年龄、性别等。
- **账户信息**：本软件无需注册或登录，没有任何账户系统。
- **设备标识符**：不收集设备 ID、MAC 地址或其他唯一标识符。
- **IP 地址**：本软件不会主动记录或存储你的 IP 地址。
- **地理位置**：不收集、不追踪你的地理位置。
- **浏览器历史记录**：不读取、不收集你的网页浏览记录。
- **个人文件内容**：不读取文档、图片、视频、聊天记录等文件的实际内容。
- **遥测数据**：不包含任何遥测、分析、埋点 SDK，不向开发者发送任何使用统计数据。
- **崩溃报告**：不自动收集或上传崩溃报告。

### 1.2 我们**不**收集的扫描相关数据

- **文件内容**：扫描过程中，文件内容仅在你的设备本地处理，**绝不会**上传到远程服务器。
- **文件列表**：扫描结果和文件列表不会发送给任何第三方或开发者。

---

## 2. 扫描过程中产生的本地数据

本软件在执行安全扫描时，会在**你的设备本地**处理以下数据：

| 数据类型 | 用途 | 存储位置 | 是否外传 |
|----------|------|----------|----------|
| 文件路径 | 定位待扫描文件 | 内存（扫描结束即释放） | 否 |
| 文件哈希值（MD5/SHA256） | 与本地黑名单比对、VirusTotal 查询（见第 3 条） | 内存（扫描结束即释放） | 仅 VirusTotal（见第 3 条） |
| 威胁检测记录 | 显示扫描历史 | Windows 注册表 (QSettings) | 否 |
| Windows Defender 扫描结果 | 通过本地 MpCmdRun.exe 获取 | 内存 | 否 |
| Defender 状态信息 | 显示实时防护状态 | 内存 | 否 |

**重要说明**：
- 以上所有数据处理均发生在你的**本地设备**上。
- 除 VirusTotal 引擎（见第 3 条）外，**没有任何数据被发送到互联网**。

## 3. VirusTotal 云引擎（唯一的外联网功能）

本软件提供可选的 **VirusTotal 云查询引擎**。此功能存在以下数据交互：

### 3.1 启用条件

VirusTotal 引擎**默认不启用**。你需要自行申请并配置 VirusTotal API 密钥后才可使用此功能。未配置 API 密钥时，此引擎在扫描中自动跳过。

### 3.2 传输的数据

当 VirusTotal 引擎启用并执行扫描时，本软件会向 `www.virustotal.com` 发送以下数据：

- **文件的 SHA256 哈希值**：从待扫描文件中计算出的 SHA256 加密哈希值（64 位十六进制字符串）。

### 3.3 **不**传输的数据

- **不**上传文件内容——只发送哈希值，而非文件本身。
- **不**发送文件名称——哈希值不包含原始文件名信息。
- **不**发送你的个人身份信息。

### 3.4 隐私影响说明

SHA256 是单向加密哈希函数，从哈希值**无法**反向推导出原始文件内容。VirusTotal 仅能反馈该哈希在其已知数据库中的匹配结果：

- 如果该文件已被他人上传过，VirusTotal 可以告知该文件是否被检测为恶意。
- 如果该文件从未被上传过，VirusTotal 将返回"未见"结果，且无法获得文件内容。

**你的责任**：使用 VirusTotal 引擎即表示你接受 VirusTotal 的隐私政策（https://support.virustotal.com/hc/en-us/articles/115002168385-Privacy-Policy）。本软件开发者不对此第三方服务的数据处理行为负责。

## 4. 本地存储的数据

以下数据存储在你的本地设备上：

| 数据 | 存储位置 | 用途 | 是否可清除 |
|------|----------|------|------------|
| 威胁检测历史 | Windows 注册表 (HKCU\Software\ZackDefender) | 记录检测到的威胁，便于查看历史 | 是（软件内提供清除功能） |
| 已隔离文件 | 本地隔离目录 | 安全隔离可疑文件 | 是（软件内提供恢复/删除功能） |
| 软件设置 | Windows 注册表 | 保存用户偏好设置 | 是 |
| 版本更新记录 | Windows 注册表 | 判断是否需要显示更新日志 | 是 |
| 调试日志 | 系统临时目录 (zackdefender_debug.log) | 记录程序启动信息，用于故障排查 | 可手动删除 |

**以上所有本地数据均不会自动发送给开发者或任何第三方**。

## 5. 我们**不**做的事情

为了清晰起见，我们明确声明本软件**不**做以下事情：

- ❌ 不收集用户身份信息
- ❌ 不追踪用户行为
- ❌ 不投放任何形式的广告
- ❌ 不将任何数据出售给第三方
- ❌ 不与第三方共享数据用于营销目的
- ❌ 不将扫描数据用于 AI 模型训练
- ❌ 不上传文件内容到任何服务器
- ❌ 不在后台进行静默数据收集
- ❌ 不包含任何遥测或分析 SDK
- ❌ 不拦截或记录网络流量内容
- ❌ 不读取你的个人文档、邮件、聊天记录
- ❌ 不修改你的浏览器设置或搜索引擎

## 6. 数据安全

### 6.1 本地数据保护

由于绝大多数数据存储在本地设备上，数据安全性主要取决于你设备的物理安全和 Windows 系统的安全性。我们建议：

- 保持 Windows 系统更新
- 使用强密码或 PIN 保护你的设备
- 不要与不可信的人共享你的设备账户

### 6.2 第三方服务的风险

VirusTotal 引擎通过 HTTPS 加密协议传输哈希数据，但网络传输本身存在固有风险。VirusTotal 收到的哈希数据受其自身的隐私政策和安全措施保护，本软件开发者对此没有控制权。

## 7. 儿童的隐私

本软件是一款安全工具，不面向儿童也并不针对儿童。本软件不收集任何用户的个人信息（无论年龄），因此不涉及儿童隐私数据的问题。

## 8. 数据保留

| 数据类型 | 保留期限 |
|----------|----------|
| 威胁检测历史 | 永久（除非你手动清除） |
| 隔离文件 | 永久（除非你手动恢复或删除） |
| 软件设置 | 永久（除非你卸载软件或手动重置） |
| 调试日志 | 每次启动时覆盖写入，仅保留最近一次启动的日志 |

卸载本软件后，Windows 注册表中的相关设置和历史记录**不**会被自动清除（这是 Windows 程序的标准行为）。如需完全清除，请使用 Windows 注册表编辑器手动删除 `HKCU\Software\ZackDefender` 项。

---

## 9. 软件更新

本软件当前不包含自动更新功能。更新管理器仅比较本地版本号，用于判断是否显示版本更新日志。它**不**连接到任何远程服务器检查新版本。
